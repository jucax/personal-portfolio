Rock Paper Scissors.

This project features a Rock-Paper-Scissors game, highlighting the Project Overview, Tools Used, and Live Links to the interactive application.

About it.

I developed a dynamic Rock-Paper-Scissors game as part of my journey in learning JavaScript, incorporating modern CSS and HTML for an engaging design.
The game allows users to choose their move while the computer selects a random move, with the result displayed immediately. It also features an automated mode where the computer plays against itself, and a real-time score tracker updates after each game.
