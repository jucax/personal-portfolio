Corre En El Tiempo.

This project features the videogame Corre En El Tiempo, it was the first videogame that I created and that allowed me to win a contest.

About it.

Corre En El Tiempo is my first video game developed in Unity, with which I won the national DinamITAM 2022 competition.
This 2D vehicle game includes various levels, a health bar, start screen, retry screen, and final screen. I created all the pixel art assets and developed the entire code for the game's functionality. This project showcases my skills in game design, software development, and digital art creation.
