Pero ¿No Me Hackean?.

This project features Pero ¿No Me Hackean? is a Full Stack web application designed to manage the users of my social project.

About it.

In this website, students can register or log in, view all the courses they have taken and those that have been offered or will be offered, watch recorded class sessions, access a calendar of activities, and have access to manuals and tutorials for various applications.
On the admin side, the application allows me to get to know my students, gather statistics, and track their activities. This project demonstrates my capabilities in web development, user management, and creating educational resources.
