Calculator.

This project features a Calculator webpage, the first project where I used HTML, CSS and JS to create a fully functional calculator similar to the iPhone's.

About it.

This is a fully functional web application inspired by the iPhone's calculator. This project helped me learn to write cleaner code using functions to handle repeated actions and to implement local storage for data persistence across page reloads.
Through developing this project, I gained valuable experience in creating reusable code, managing application state, and enhancing user experience with persistent data.
